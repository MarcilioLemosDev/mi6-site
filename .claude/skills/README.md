# Skills do projeto

- `frontend-design/`: skill oficial da Anthropic para interfaces com alta
  qualidade de design, sem estética genérica de IA. Fonte:
  https://github.com/anthropics/claude-code/tree/main/plugins/frontend-design
  (vendorizada em 11/06/2026). Em conflito, a seção "Direção de arte" do
  CLAUDE.md prevalece (paleta Union Jack fixa, títulos em serif, proibições).

Para aprofundar animação GSAP em fases futuras, existe a skill oficial da
GreenSock: https://github.com/greensock/gsap-skills
(instalável com `npx skills add https://github.com/greensock/gsap-skills`).
